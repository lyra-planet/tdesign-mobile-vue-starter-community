# 设置 Setting

- 组成：General 设置、深色模式（Black Mode）
- 路由：`/my/settings`（由“我的”页面入口进入）

## 功能与交互

- 通用设置项（通知、语言、清理缓存等示例）
- 深色模式切换（基于 `useTheme()`），返回后全局样式即时生效
