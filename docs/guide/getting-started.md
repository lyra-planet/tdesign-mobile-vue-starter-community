# 快速开始

## 环境要求

- Node.js 版本 >= 20.9.0
- pnpm 版本 >= 10

## 安装

```bash
# 克隆项目
git clone https://cnb.cool/hikari0818/tdesign-mobile-vue-starter-community.git

# 进入项目目录
cd tdesign-mobile-vue-starter-community

# 安装依赖
pnpm install
```

## 开发

```bash
# 启动开发服务器
pnpm dev
```

## 构建

```bash
# 构建生产版本
pnpm build

# 预览构建结果
pnpm preview
```
